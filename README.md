# E-Commerce Web Scraping & Data Extraction Pipeline

## 📌 Project Overview
This project demonstrates an automated web scraping pipeline built with Python. It extracts product data (titles, prices, ratings, availability) from an e-commerce platform, cleans the data, and exports it into multiple business-ready formats.

## 🛠️ Tools & Technologies
- **Language**: Python
- **Web Scraping**: BeautifulSoup (`bs4`), `requests`
- **Data Processing**: Pandas, NumPy
- **Data Visualization**: Matplotlib, Seaborn

## ⚙️ Key Features
1. **Automated Extraction**: Scrapes hundreds of products instantly.
2. **Data Transformation**: Converts string currencies to numeric floats and normalizes ratings for statistical analysis.
3. **Timestamping**: Adds exact scraping time for historical price tracking.
4. **Multi-Format Export**: Generates `.xlsx`, `.csv`, and `.json` files to suit different client needs.
5. **Quick Insights**: Includes visualizations analyzing price distributions and price-to-rating correlations.

## 📈 Real-World Use Cases
- Competitor price monitoring.
- Market research and trend analysis.
- Building custom datasets for Machine Learning.
